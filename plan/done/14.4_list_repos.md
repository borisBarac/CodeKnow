    github_ssh_url: str
    slug: str
    commit_hash: str
    built_at: str
    node_count: int
    edge_count: int
    community_count: int

    registered: bool = True
    orphan: bool = False

    page: int
    page_size: int

## Risk notes

- **Performance**: `health_check=true` calls `load_graph()` per repo which parses multi-MB JSON files. Kept opt-in to avoid default cost.
- **`RepoMetadata` with `extra="allow"`**: New fields in `metadata.json` will pass through without breaking the model.
- **`orphan_registrations`**: Currently collected but not included in the typed `ListReposResponse`. Consider adding as optional field or returning separately.
